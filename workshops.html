Introduction to Markdown

A one-hour workshop on the fundamentals of the markdown language.

Designed to introduce students to the markdown language. My goal for this workshop is to help familiarize students with Markdown to help develop README files to document their portfolio projects.

Introduction to Database Programming

A one-hour introduction to database programming (triggers, stored procedures, functions).

At Tech Elevator, we teach students how to query data, insert data, and create/modify tables using SQL, but as we are not a database bootcamp, we don't get into some of the procedural programming. When we get to our database portion of the program, some students really fall in love with databases, and I put this workshop together to show students you can do more than just query data.
