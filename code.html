Database Setup

A script I created in Excel to generate my class MySQL schemas, grant permissions, and load data to their schemas.

I've ported it over to Google Sheets. My goal was to paste my roster and have everything generated for me.
This strips the domain from the username, creates an account with the same username (using Windows authentication/single sign on), create a new schema for the user, grant them permissions, and populate it, and give read/execute access to common databases. There are a few other tabs to clean up student accounts at the end of the semester and refresh instructor and tutor accounts.
Basically, it's a great way to use Excel's string manipulation.

Generating Diverse Datasets
